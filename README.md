# 微信自动回复机器人

#### 介绍
微信自动回复机器人，有三个机器人可供选择，可在界面进行配置，可定时提醒，bs端程序，基于C#winfrom程序

#### 安装教程

直接打开\Setup1\setup.exe即可安装 需要安装的环境已经集成
